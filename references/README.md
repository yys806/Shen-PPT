# Shen-PPT References

This directory contains the runtime assets that Shen-PPT needs to produce stable, repeatable PPT decks.

## Contents

| Path | Purpose |
|---|---|
| `parameter-spec.md` | fixed geometry, typography, icon, density, no-animation, and QA parameters |
| `highest-references/orangepi-defense-final-v9-20260607/` | highest-quality reference package for judging output quality |
| `orangepi-defense-final-v9-20260607/` | compatibility mirror of the highest reference package |
| `style-samples-v2-20260606/sample-decks/` | 15 editable four-page sample decks plus overview images |
| `style-samples-v2-20260606/sample-deck-map.json` | stable slug-to-template path map |

## Rules

- PPTX templates, preview PNGs, contact sheets, and reference decks belong here, not in the repository root.
- The OrangePi deck is a quality reference, not a reusable template.
- Style sample decks are editable base decks. When selected, they should be filled and extended directly.
- Real user material always overrides sample content.
